# professional_world 初始化
## 克隆项目，然后进入项目目录
`cd professional_world/`
## 1.是否安装了rvm ruby 环境
`rvm list` <br/>   
`ruby -v ` <br/>   
#=> ruby 2.3.4p301 (2017-03-30 revision 58214) [x86_64-linux] <br/>   
`rvm gemset list` <br/>   
#=>professional_world_pro
## 2.然后安装gem
`gem install bundler <br/>   bundle install`
